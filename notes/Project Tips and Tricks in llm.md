## Structure of creating a machine learning project 
1. **load** dataset 
2. **preprocessing** on dataset (is have null values ? , duplicated values ? , handle missing values by fill with empty or mean , Encoding categorical dataset and so on ... ) 
3.  **split** data into training and testing 
4. **test** training accuracy and testing accuracy 
5. **create** a test system
