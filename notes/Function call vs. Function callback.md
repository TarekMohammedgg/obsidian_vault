#technical #basics 

function call (Function()): 
انا بستدعي الفانكشن و بنفزها دلوقتي . 


function callback (Function ) : 
انا بمرر الفانكشن دي لحاجه تاني عشان الحاجه دي تبقي تتنفزها بس مش دلوقتي بعدين لما تحتاجها . 


----
مثال بسيط : 
احمد عنده لعبة بيلعبها لما يحب يلعبها دلوقتي يبقي ده اسمه function call (Function ())
و لكن لما احمد يقول لاخته لو حبيتي تلعبي ابقي استعملي اللعبة دي و يبقي اسمه function callback (Function)
