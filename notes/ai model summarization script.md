كل المعلومات دي  
بنحطها في شكل  **JSON**  قبل ما نرفع البيانات دي للـ backend  
و ده بيتم باستخدام  مكتبة اسمها **Pydantic**  في بايثون بتخلينا نحدد شكل البيانات  و بنعملها validation  
يعني بنقول كل معلومة ال data type بتاعتها ايه و description بسيط عنها 
و بعدين بدخله علي summarisation template 
عشان اطبق ال **prompt engineering**   من خلال اني بديله  few shots guides و  ال pedantic structure  model و ده زي تعليمات الموديل بيشمي  عليها أثناء عمل ال summarisation ، مكنتش محتاج اني اعمل fine tuning أو استخدم rag 
و بعدين بيروح ال summarisation  كجزء لل report الي بيظهر للدكتور و بعدين يقدر يتكلم مع  موديل خاص للدكتور
