## Plugins 
1. image converter - to edit on the original image 
2. lapel  - for show the header number (h1 , h2, ... , h6)
3. image tool kit  - to view and zoom in and out in image 
4. data view 
5. Journals 
6. Fast Text color 
7. Latex Suite
8. Space Repetition 
9. Excalidraw   - make the obsidian take more time  when start up 
10. Obsidian  Git - to create a backup for your vault 
11. Obsidian Sync - awesome for syncing your notes or  vaults with multiple devices , you can use **syncthing** alternate obsidian sync 
12. Global search and replace plugin [here](https://github.com/MahmoudFawzyKhalil/obsidian-global-search-and-replace) : to solve problem delete   , by type the name of tag and replace it with space
13. linter [video for explain]([https://youtu.be/pAALEE6Tc3E](https://youtu.be/pAALEE6Tc3E "Share link")): for formatting on the bulk of notes like add property , remove it or update . 
14. Khoj : chat bot can access and make conversation about your notes .
15. File Explorer Note Count .
16. local Backup
17. inscribe : for auto complete your notes.
18. various complements : to autocorrect your notes.
19. beautitab : to create beautiful dashboard when open new tab.
